# 📌 ADRs — Architecture Decision Records

> **Status:** ⏳ Pendente

ADRs documentam decisões arquiteturais importantes: o contexto, a decisão tomada e as consequências.

!!! note "Em construção"
    Os ADRs serão criados durante a fase de arquitetura.

## Formato de um ADR

```
# ADR-000 — Título da Decisão

**Status:** Proposto | Aceito | Depreciado
**Data:** YYYY-MM-DD

## Contexto
Por que essa decisão precisou ser tomada?

## Decisão
O que foi decidido?

## Consequências
Quais são os impactos positivos e negativos?
```
